# 🧙‍♂️ 哈利波特百科全书网站 — 总体规划

> **愿景**：打造中文互联网最权威、最全面、最美观的哈利波特百科全书网站，成为中文搜索"哈利波特"排名第一的网站。

---

## 📋 一、项目概况

- **项目名称**：哈利波特的魔法世界（Harry Potter's Wizarding World）
- **定位**：哈利波特电影×原著百科全书 + 每日新闻资讯站
- **技术栈**：React 18 + Vite 5 + React Router 6 + 纯 CSS
- **域名建议**：owlpost.cn / harrypotter.wiki.cn
- **目标用户**：中文哈利波特粉丝（哈迷）

#### 📸 人物生平介绍与剧照规范

##### 生平介绍（biography 字段）

每位人物必须有详细的生平介绍，不再只是一两句简介。生平介绍应当是一篇完整的小传记，内容包含：

1. **早年生活**：出生背景、家庭环境、童年经历
2. **霍格沃茨时期**：入学分院、学业表现、重要友谊与冲突、参与的关键事件
3. **成年/战后生活**：毕业后的职业发展、在魔法战争中的角色、最终结局
4. **性格与评价**：性格特点分析、其他角色对其的评价、在整个故事中的意义
5. **演员信息**：饰演该角色的演员姓名、演员简介

> **字数要求**：
> - 主要角色（哈利、赫敏、罗恩、邓布利多、斯内普、伏地魔等）：**不少于 800 字**
> - 重要配角（金妮、马尔福、小天狼星、卢娜、麦格等）：**不少于 500 字**
> - 其他角色：**不少于 300 字**

##### 剧照规范（photos 字段）

每位人物需附上高质量的电影剧照，在人物详情页以「剧照画廊」形式展示。

> **剧照数量要求**：
> - 🌟 **主要演员**（哈利·波特 / 赫敏·格兰杰 / 罗恩·韦斯莱 / 阿不思·邓布利多 / 西弗勒斯·斯内普 / 伏地魔）：**至少 10 张剧照**
> - ⭐ **重要配角**（金妮·韦斯莱 / 德拉科·马尔福 / 小天狼星·布莱克 / 卢娜·洛夫古德 / 米勒娃·麦格 / 海格 / 纳威·隆巴顿等）：**至少 6 张剧照**
> - 其他角色：**至少 3 张剧照**

> **剧照选取原则**：
> 1. 覆盖不同电影——尽量从该角色出场的不同电影中各选取代表性剧照
> 2. 展现不同状态——包含日常、战斗、情感、喜剧等不同场景
> 3. 标注信息——每张剧照需标注：
>    - `src`：图片路径（统一存放在 `public/images/characters/{人物id}/` 目录下）
>    - `caption`：简短的场景描述（如"哈利在霍格沃茨大战中直面伏地魔"）
>    - `movie`：出自哪部电影（如"哈利·波特与死亡圣器(下)"）
> 4. 图片格式与尺寸：
>    - 使用 AI 生成的高质量剧照风格图片（因版权原因不使用真实剧照）
>    - 推荐尺寸：宽 800px 以上，16:9 或 3:2 比例
>    - 格式：WebP 或 JPG，单张不超过 200KB

##### 数据结构升级

在 `src/data/characters.js` 中，每个人物对象需新增以下字段：

```javascript
{
  id: 'harry-potter',
  name: '哈利·波特',
  // ... 原有字段保持不变 ...

  // 新增：详细生平介绍（替代原来的短 description）
  biography: `
    哈利·波特于1980年7月31日出生于戈德里克山谷...（详细传记，至少800字）
  `,

  // 新增：饰演该角色的演员
  actor: 'Daniel Radcliffe（丹尼尔·雷德克里夫）',

  // 新增：剧照数组
  photos: [
    {
      src: '/images/characters/harry-potter/01.webp',
      caption: '11岁的哈利第一次来到对角巷，对魔法世界充满好奇',
      movie: '哈利·波特与魔法石'
    },
    {
      src: '/images/characters/harry-potter/02.webp',
      caption: '哈利骑着火弩箭参加魁地奇比赛',
      movie: '哈利·波特与阿兹卡班的囚徒'
    },
    // ... 主要角色至少10张
  ],
}
```

##### 页面展示升级

`CharacterDetail.jsx` 人物详情页需新增以下 UI 模块：

1. **生平传记区**：将 biography 内容以排版优美的长文形式展示，替代原来的短简介。支持分段落显示，配合首字下沉等排版美化。
2. **演员信息区**：展示演员姓名和简介。
3. **剧照画廊区**：
   - 以网格/瀑布流布局展示所有剧照缩略图
   - 点击缩略图可弹出灯箱（Lightbox）查看大图
   - 灯箱内支持左右切换浏览
   - 每张图下方显示 caption 和 movie 信息
   - 响应式布局：桌面端 3-4 列，平板 2 列，手机 1-2 列
